# v0 Seed Benchmark

This folder contains the first generated seed benchmark for the Traditional Chinese long-term conversational memory project.

## Files

- `personas.jsonl`: 10 synthetic personas across 5 Taiwan-context scenario types.
- `dialogues.jsonl`: 100 user turns, 10 sessions per persona.
- `gold_events.jsonl`: 100 gold atomic events derived from the designed timelines.
- `gold_update_relations.jsonl`: 40 gold update relations.
- `qa.jsonl`: 100 QA items with gold answers, evidence turn ids, valid time, abstention flags, and optional update-relation labels.

## Construction

The dataset is generated by:

```powershell
$env:PYTHONPATH="src"
C:\Users\o1000\anaconda3\envs\fuckyou\python.exe -m event_memory.cli generate-dataset
```

The generation flow is:

```text
persona seed
-> gold event timeline
-> dialogue turns
-> gold update relations
-> QA items with gold annotations
-> validation
```

## Current Status

This version is designed to validate the schema and experiment pipeline. It is intentionally template-like. The next dataset step is to use an LLM to rewrite the dialogue turns into more natural Traditional Chinese while preserving all gold evidence turn ids and labels.
